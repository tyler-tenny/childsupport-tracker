## Child Support Case Tracker
Full-stack case management app with Java, Spring Boot, PostgreSQL, and Angular.

## About
Built to get familiar with full-stack Java, particularly Spring Boot. The app manages cases, participants and payment tracking through a normalized relational database with JWT authorization and Angular frontend. The theming comes from an opportunity I was pursuing at the time.

## Prerequisites
- Java 21
- PostgreSQL
- Node.js and Angular CLI
 
Use 'application.properties.example', fill in your values as needed
